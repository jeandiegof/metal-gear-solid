Creates the points that constitute a Line object.

This function creates a *line that starts at the origin with size equals to lenght. The inclination of the line is determined by angle.

Parameters
origin[in]Position to the first Point of the Line.
angle[in]Angle of the Line.
lenght[in]Line's lenght in pixels.
line[out]A pointer to a Line.
