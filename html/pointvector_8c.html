Inserts #value at the given #index.

If the index is greater than the #vector->size, this function should expand the vector until it is big enough to contain the index and set the value at that index. It should zero-fill all values in between. #vector->size should be incremented accordingly.

void VectorPointResizeIfFull ( VectorPoint *  vector)

Doubles the underlying data array capacity if #vector->size >= #vector->capacity.

Changing the size of the array is expensive, so in order to minimize the number of times it's resized, the capacity is doubled each time.
