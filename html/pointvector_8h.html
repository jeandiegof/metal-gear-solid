static void VectorPointResizeIfFull ( VectorPoint *  vector)
static

Increase the underlying data array capacity if vector->size >= vector->capacity.

This function expand data array capacity by allocating a number of slots equal to VECTOR_POINT_DEFAULT_CAPACITY.

1 vector->capacity += VECTOR_POINT_DEFAULT_CAPACITY;
2 vector->data = realloc(vector->data, sizeof(Point) * vector->capacity);
