typedef struct Point Point

Abstract struct for a point in console.

Implements a point in x, y coordinate system.

Although coordinates that a console can represet are always positive, the variables #x and #y are from type int16_t. Meaning that they could be used to store negative values. This feature can be useful in some situations, e.g. storage of some offset value.
